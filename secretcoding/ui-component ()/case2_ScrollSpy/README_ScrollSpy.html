문제 풀기 방식 :

  1. 레포지토리를 clone
  2. 터미널에서 각 문제 폴더 디렉토리로 이동하여 npm install로 의존성을 설치
  3. package.json을 참고하여, 명시된 scripts 명령어로 개발서버 실행.
