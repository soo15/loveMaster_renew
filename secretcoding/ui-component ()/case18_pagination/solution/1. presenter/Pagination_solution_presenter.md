​
시작지점부터 한페이지에 보여질 게시물 수를 보여준다
- 1 page ) 0번부터 5(한페지에 보여줄숫자)까지의 게시물을 보여주면된다. start : 23*(1페이지-1) * limit
- 2 page ) (2-1) * 5
- 3 page ) (3-1) * 5
각 시작지점은 위와 같고 5개의 게시물씩 보여주면 된다.

  var result = "";
