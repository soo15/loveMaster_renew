플레스(Flex)를 왜 사용하나요?

  • float 속성을 사용하지 않고 가로배치한다.
  • 반응형 레이아웃을 만들기 쉽다.
  • 자식요소의 위치 잡기(Positioning)를 훨씬 쉽게 한다.
  • 브라우저에 보여지는 HTML 요소의 순서를 HTML 코드를 수정하지 않고 가능하다.

Flex 속성

  • display: flex | inline-flex
  • flex-direction: row | column
  • flex-wrap: wrap | nowrap | wrap-reverse
  • flex-basis: 길이
  • justify-content: flex-start | flex-end | center | space-between | space-around
  • align-self: auto | flex-start | flex-end | center | baseline | stretch
  • align-items: flex-start | flex-end | center | baseline | stretch
  • align-content: flex-start | flex-end | center | space-between | space-around | stretch
  • flex-grow: 숫자(정수)
  • flex-shrink: 숫자(정수)
  • flex: 숫자(정수)
  • order: 숫자(정수)

부모 요소와 자식 요소에 정의하는 Flex 속성 구분

  • 부모요소에 쓰는 속성: display:flex, flex-direction, flex-wrap, justify-content, align-items, align-content
  • 자식요소에 쓰는 속성: flex, flex-grow, flex-shrink, flex-basis, order, align-self
